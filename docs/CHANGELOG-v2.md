# v2.0 — Имена пользователей в попапе реакций

## Что нового

При наведении на кнопку реакции показывается тултип с именами пользователей:

- "Прореагировали Харизма, Самочка и Катя Рустанова."
- Если больше 3 — "Прореагировали Харизма, Самочка и ещё 5."

## Изменения в файлах

### Worker

| Файл | Изменения |
|------|-----------|
| `reactions-do-v2.js` | Новый эндпоинт `GET /voters?emoji=👍` — возвращает массив user_id кто проголосовал за конкретный эмодзи |
| `index-v2.js` | Проксирует GET с параметрами `post_id` и `emoji` на DO |

### Widget

| Файл | Изменения |
|------|-----------|
| `forum-reactions-v2.js` | Добавлены: `NAME_CACHE`, `fetchNames()`, `showTooltip()`, обработчики `mouseenter`/`mouseleave` |

### Нагрузка

| Что | Было | Стало |
|-----|------|-------|
| Workers requests | 1 GET на страницу | 0 (всё на клиенте) |
| DO requests | 6 на страницу (30 постов / 5) | +1 при hover (редко) |
| Forum API | 0 | +1 при hover (форум отдаёт бесплатно) |

## Как подключить

1. Замените `reactions-do.js` на `reactions-do-v2.js`
2. Замените `index.js` на `index-v2.js`
3. Замените виджет на `forum-reactions-v2.js`
4. Перезадеплойте Worker: `wrangler deploy`

## Откат

Верните оригинальные файлы без `-v2` и перезадеплойте.
